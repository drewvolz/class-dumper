# Class Dumper

## About

A macOS app around the [class-dump][1] utility for examining Objective-C runtime information stored in Mach-O files.

### Features

- Invoking `class-dump` from a Finder file picker
- Saving results to the Documents directory
- Reloading saved dumps in the file browser
- Removing individual dumps / resetting all content
- Showing the analyzed Mach-O file on-disk via Finder
- Filematching per-dump from search
